The atom-atom polarizability is a chemical quantity defined for the isolated molecular graph. The eigenfunctions (molecular orbitals), ∣ψi⟩| \psi_i \rangle∣ψ​i​​⟩, and eigenenergies, EiE_iE​i​​, are defined by the Schrödinger equation:

H∣ψi⟩=Ei∣ψi⟩ H |\psi_i \rangle = E_i |\psi_i \rangle H∣ψ​i​​⟩=E​i​​∣ψ​i​​⟩

Each orbital is a vector with an entry for each node (or atom) in the graph:

∣ψi⟩=(ψi1,ψi2,…,ψiN) |\psi_i \rangle = (\psi_i^1, \psi_i^2, \ldots, \psi_i^N) ∣ψ​i​​⟩=(ψ​i​1​​,ψ​i​2​​,…,ψ​i​N​​)

We fill those orbitals from the bottom up, with two electrons per orbital. The filling is then directly given by the highest occupied orbital, mmm.

The atom-atom polarizability between node rrr and sss is then given by,

πr,s=−4∑j=1m∑k=m+1N(ψrj)∗ψsj(ψrk)∗ψskEk−Ej. \pi_{r,s} = - 4 \sum_{j=1}^m \sum_{k=m+1}^N \frac{(\psi_{r}^j)^* \psi_{s}^j (\psi_{r}^k)^* \psi_{s}^k }{E_k - E_j}. π​r,s​​=−4​j=1​∑​m​​​k=m+1​∑​N​​​E​k​​−E​j​​​​(ψ​r​j​​)​∗​​ψ​s​j​​(ψ​r​k​​)​∗​​ψ​s​k​​​​.

The expression is taken from Stuyver et al. but generalized to handling complex orbitals.

The polarisability widget

The highest occupied orbital can be chosen from a drop down menu. Note that the polarizability is ill-defined when the highest occupied orbital and the lowest unoccupied orbital have the same energy. We circumvent this problem by automatically changing such invalid choices of the highest occupied orbital to the nearest valid one.

Additionally choosing one atom (or node) from the other drop down menu, displays the atom-atom polarizability for this node with respect to all nodes. So choosing e.g. atom number rrr displays πr,s\pi_{r,s}π​r,s​​ for all nodes sss.

The polarizability is drawn using filled circles where the polarizability is proportional to the area of the circle. The chosen atom rrr has been filled using a netural color, leaving only a ring. The sign is encoded in the color of the orbital with red signifying a positive value and blue (cyan) a negative value.
