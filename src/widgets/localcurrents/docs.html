The Local Currents

The local current describe the local particle current along each of the links of the graph arising when a particle travels from one or more bands to others.

A band supports particle states at many different energies, and we can separate the local currents from each of those energies, EEE, assuming that particles only originate from a single band, and not from any others.

The local current can be expressed through the lesser Green's function, which in the case of single source band, sss, takes the form,

The Local Currents Widget

The widget allow you to choose one input band from a menu toggled by clicking the "?" icon. The incoming energy can be tuned using the slider at the bottom of the widget.

The local currents along the links in the graph is shown as directed arrows. The size of the arrow reflects the size of the current and the orientation define its direction. The arrows are colored blue when pointing in directions [45°,225°][45\degree, 225\degree][45°,225°] and red otherwise.
