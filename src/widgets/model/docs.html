The Nodes

The editor shows the nodes as circles. By selecting a circle, you can edit the node weight. The size of the node weight is reflected in the radius of the circle, and the sign of the node weight is reflected in the shading of the node.

The Links

If you have toggled the link button, links can be created by selecting two nodes right after each other. If there already is a link between the two nodes, the editor automatically selects it. Links are created with a default strength of −1-1−1, reflecting the fact that kinetic energy overlaps usually are negative. The link strength can be changed to other negative values, with larger values represented as thicker lines.

In addition, the phase of the link can be manipulated using a circular slider. A phase of 180° corresponds to changing the sign of the link to positive. The phase is shown as a directed arrow, and the 180° phase is shown as a line cutting across the link making it resemble a "+". This is because both a negative and a positive link strength does not signify a preferred direction of the link.

The direction of the link can also be changed by pressing the "flip" button. This flips the direction of the link, and correspond to changing the phase, ϕ\phiϕ to 360deg−ϕ360\deg - \phi360deg−ϕ.

The V.AL Inspector

You can get a quick overview of the different node weights and link strengths by selecting the "V.AL" button. This show the values overlaid on the graph.

Bands

The graph can be connected to wide bands. These bands are labeled and drawn as yellow circles. This is to signify that they represent idealized conductors, and may in experiments be composed of gold or other metals. To signify that the bands are separated from the graph, the links between the bands and the graph are drawn as dashed lines. Despite the fact that those links can be modified in the same way as links between normal nodes, the links to bands are treated separately by the widgets.

You can learn more about the way bands are included in the calculations by reading the documentation for the "Transmission" widget or the Scattering State" widget.
